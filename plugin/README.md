# As a plugin for LMMS

```
cd lmms/plugins
ln -s [LRCScroll.lmms/plugin] LRCScroll

# cmake adjustment: edit lmms/cmake/modules/PluginList.cmake
# Add LRCScroll in the list
```

## Scroller Features:

 * [ ] Ctrl+Plus: Zoom in text
 * [ ] Ctrl+Minus: Zoom out text
 * [ ] Left Arrow, Right Arrow: alignment
 * [ ] F9/F10 switch between editor and scroller
